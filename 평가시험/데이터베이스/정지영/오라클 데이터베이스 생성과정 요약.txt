오라클 데이터베이스 생성과정

1.  Oracle -> Database Configuration Assistant 실행
->데이터베이스 생성 -> 디폴트 선택
2.  전역 데이터베이스 이름과 SID를 입력(SID까먹으면안됨)
3.  Enterprise Manager 구성을 체크(성능관리가 필요하다면 체크 아니면 해제)
4.  새 데이터베이스에 대한 비밀번호 설정
5.  Oracle-Managed Files 사용을 체크한 후,
6.  데이터베이스 플래시 복구영역 지정(디폴트)
7.  샘플 스키마선택(선택사항)
8.  메모리 디폴트선택 -> 유니코드(AL32UTF9)사용을 체크
9. 데이터베이스 저장영역 기본설정으로
10. 생성완료
