1. Database Configuration Assistant를 실행한다.
2. 데이트베이스 생성( 디폴트 ) 선택한다.
3. 범용 또는 트랜잭션 처리 선택한다.
4. 전역 데이터베이스 이름과 SID를 설정한다.
5. Enterprise Manager 구성한다.
6. 비밀번호를 설정한다.
7. 저장 영역 유형을 파일 시스템으로 선택한다.
8. Oracle-Managed Files 사용에 체크하고 데이터베이스 영역을 설정한다.
9. 플래시 복구영역을 설정한다.
10. 샘플스키마를 선택한다.
11. 문자집합은 한자를 사용할 경우 Unicode(AL32UTF8)로 설정한다.
12. 접속모드는 동시접속자가 많다면 공유서버모드, 테스트 용도라면 전용 서버 모드 선택한다.
13. 데이터베이스 저장 영역을 구성하는 제어파일, 데이터파일, 리두로그 그룹을 확인한다.
14. 데이터베이스 생성 체크하고 완료 누른다.


1.  DB ( 테이블 스페이스 ) 생성
2. 사용자 생성 ( 1에서 만들어 놓은 DB를 지정 )
